Создаем приложение которое будет подключаться к серверу в интернете, по URL.
Данные будут загружаться один раз, не синхронизируясь при изменениях.
С сервера будет получать список котов и сменем, породой и изображением и др.
Получив этот json файл, мы будем загружать из него данные в локальную базу данных и
из базы данных помещать этих котов в список и отображать на экране.
